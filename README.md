# Mangchi_Final - 스프링 REST-ful 프로젝트
우리동네 대여 서비스 망치! 나눔게시판 서버    

<hr />

# 주요 기능    
* Ajax를 활용한 서버-클라이언트 MSA 방식의 통신     
* 원페이지 모달 방식 뷰    
* 이미지 포토카드 형식의 게시판 리스트      
* Summernote Api를 활용한 에디터    
* 무한스크롤과 댓글 페이징      
* 검색       
     
# 넣으려다 삽질하고 다음 프로젝트로 이전한 기능    
* FireBase Api를 활용한 푸시알림

<hr />

# 사용 기술    
* VIEW : JSP     
* SERVER : JAVA8    
* DATABASE : MySQL    
* WAS : Tomcat8    
* 배포 : AWS    
* Framework : Spring Legacy   
* Lib : Maven
